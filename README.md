# kakeibo

35日（5週間）を一周期とする家計簿アプリを作りたい！

メリットは以下！

- 無理なく出費を抑えることができる
- 年に2回ボーナス月がやってくる
- ボーナス月で旅行・豪華ディナーが楽しめる！

35日で1周期の**35カケイボ**を使って楽しく無理なく家計簿を付けよう！
